# 戦略文書の議論プログラム (Strategy Document Discussion Program)

このプログラムは、会社の戦略文書について3人のペルソナが議論を行うためのアプリケーションです。

## 機能
- 3人のペルソナによる戦略文書の分析と議論
- 各ペルソナの個性的な視点と立場を反映
- リアルタイムのチャット形式での議論表示
- ユーザーによる議論の制御（開始・停止）

## 技術スタック
- バックエンド: FastAPI
- フロントエンド: React + TypeScript + Tailwind CSS
- UI コンポーネント: shadcn/ui

## ディレクトリ構造
